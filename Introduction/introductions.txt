<h3><ul>Example Explained</ul></h3>
<ul><li>The <font color="red">&lthtml&gt</font> element is the root element of an HTML page
<li>The <font color="red">&lthead&gt</font> element contains meta information about the document
<li>The <font color="red">&lttitle&gt</font> element specifies a title for the document
<li>The <font color="red">&ltbody&gt</font> element contains the visible page content
<li>The <font color="red">&lth1&gt</font> element defines a large heading
<li>The <font color="red">&ltp&gt</font> element defines a paragraph</ul><br>
